# 量子探索之旅 ⚛️

一个交互式的量子力学教学网站，通过 Canvas 动画模拟和可编辑的 Markdown 文档，带领学生从经典物理的困境走向现代量子应用。

> 📚 **项目名称**：Quantum Explorer（量子探索之旅）

## 🚀 快速开始

### 方法一：本地服务器（推荐）
```bash
# 启动项目内置本地服务器
node scripts/local-server.js

# 访问 http://localhost:8000
```

### 方法二：静态浏览
双击 `index.html` 或使用 `python -m http.server 8000` 也可浏览页面，但管理员上传本地文件无法落盘保存。

### 管理员登录
1. 点击右上角 ⚙️ 图标
2. 输入密码：`zjuphy`
3. 登录后可编辑所有页面内容

### 编辑教学内容
1. 以管理员身份登录
2. 进入任意教学页面
3. 点击「✏️ 编辑」按钮，打开内容与资源编辑器
4. **文本** Tab：使用 Markdown 编辑，左侧编辑、右侧预览
5. **链接 / 图片 / 视频 / 文档** Tab：管理多媒体资源
6. 点击「保存文本」，内容写入 `admin_uploads/content-data.json`

### 迁移到另一台电脑
1. 复制整个项目目录（务必包含 `admin_uploads/`）
2. 在新电脑运行：`node scripts/local-server.js`
3. 所有已编辑内容和上传文件将正常显示
