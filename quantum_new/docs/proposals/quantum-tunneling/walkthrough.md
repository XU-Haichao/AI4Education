# 薛定谔方程势垒隧穿模拟改进报告

本文档记录了对"薛定谔方程"互动页面中势垒隧穿模拟功能的重大改进。

## 1. 核心改进

### 1.1 物理模型的严格化
我们摒弃了原有的FFT分裂算子方法（存在周期性边界伪影）和基于自由粒子近似的展开方法，转而实现了**全解析的散射态时间演化**方案。

- **算法原理**：将初始高斯波包严格投影到含势垒哈密顿量的本征态（散射态）上，通过时间演化算符 $e^{-iEt/\hbar}$ 获得任意时刻的波函数。
- **重建基底**：
  - **传播态 ($E > V$)**：由平面波和反射/透射波组成。
  - **隧穿态 ($E < V$)**：在势垒内部由双曲函数 ($\cosh, \sinh$) 描述的衰减波。

### 1.2 数值稳定性的突破
在势垒内部波函数的计算中，我们解决了一个严重的数值爆炸问题。

- **问题**：在深势垒（$\kappa L \gg 1$）情况下，从入射侧直接传播指数增长项 $e^{\kappa x}$ 会导致归一化系数爆炸（达到 $10^{15}$ 量级）。
- **解决方案**：采用**反向传播法**。对于势垒内的波函数，我们从透射侧（Region III，数值微小）向左反向推导。利用 Region III 边界条件的连续性，确保了在势垒内部计算的数值稳定性。

```javascript
// 示例：势垒内部稳定计算（从右边界 b 向左传播）
// psi(x) = psi(b) [ cosh(k*dx) - i(k/kappa)sinh(k*dx) ]
// 避免了直接计算 e^{kappa*x} 带来的溢出
```

### 1.3 物理公式修正
我们修复了原代码中关于隧穿系数的公式错误：
- 修正了 $D(k)$ 分母项虚部符号错误。
- 修正了反射系数 $r(k)$ 在隧穿情形下的相位因子。
- 确保了在 $E=V$ 临界点附近的数值平滑过渡。

## 2. 参数与视觉优化

### 2.1 模拟参数调整
- **模拟区域**：调整为 $x \in [0, 35]$。
- **初始波包**：
  - 位置：$x_0 = 5.0$（避免起始点的吉布斯截断误差）。
  - 宽度：$\sigma = 1.5$。
- **势垒参数**：
  - 位置：$x \in [17.5, 18.0]$。
  - 宽度：$L = 0.5$（减小宽度以显著增加可视的透射概率）。
  - 高度：$V = 15.0$。

### 2.2 动画体验
- **速度控制**：全局时间步长 `dt` 从 0.05 降低至 0.025，使高动量波包的移动速度更符合视觉观察习惯。
- **自动重置**：实现了智能重置逻辑。当波包的绝大部分概率密度（>90%）离开模拟可视区域（无论是透射还是反射）时，动画自动重置。

## 3. 结果验证

现在，势垒隧穿模式展示了符合量子力学预期的所有特征：
1. **波包分裂**：遇到势垒后，波包清晰地分裂为透射波包和反射波包。
2. **守恒律**：在波包离开边界前，总概率归一化系数严格保持为 1.000。
3. **隧穿效应**：即使在平均能量低于势垒高度时，也能清晰看到非零的透射概率。
