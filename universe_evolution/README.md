# COSMOS | 宇宙演化史可视化 🌌

一个基于 Three.js 和 WebGL 着色器的交互式宇宙演化可视化教学网站，从大爆炸到现代宇宙，带领学生体验 138 亿年的宇宙历程。

---

## ✨ 功能特点

### 🎓 教学内容

通过可拖动的时间轴，展示宇宙演化的五大阶段：

| 阶段 | 时间范围 | 关键事件 |
|------|---------|----------|
| 🔥 极早期宇宙 | 0 ~ 10⁻³² s | 普朗克时期、大统一时期、宇宙暴涨 |
| ⚡ 粒子形成 | 10⁻³² s ~ 10 s | 夸克-胶子等离子体、轻子时期、正反物质湮灭 |
| ⚛️ 物质诞生 | 10 s ~ 380,000 yr | 太初核合成、光子退耦、宇宙微波背景辐射 |
| 🌟 结构形成 | 380,000 yr ~ 4 亿 yr | 宇宙黑暗时代、第一批恒星、再电离 |
| 🌀 星系形成与演化 | 4 亿 yr ~ 138 亿 yr | 星系合并、恒星演化、太阳系形成 |

### 🎮 交互功能
- 🕹️ **时间轴拖动**：拖动底部时间轴，自由浏览任意宇宙时期
- ▶️ **自动播放**：播放/暂停/重播，支持 0.5×/1×/2× 倍速
- 🌡️ **实时数据**：显示当前时期的宇宙年龄与温度
- 📖 **阶段文档**：右侧面板展示每个时期的详细物理描述
- 🎆 **3D 可视化**：每个演化阶段配有独立的 Three.js 粒子/着色器动画

### ✏️ 管理员编辑
- 🔐 管理员模式：编辑每个时期的描述文档
- 📁 上传附件：支持图片、视频、文档上传
- 🔗 添加外部链接
- 💾 数据保存至 `admin_uploads/content.json`

### 🎨 设计特色
- 🌑 全屏沉浸式深空界面
- ✨ WebGL 着色器驱动的粒子特效
- 🎬 平滑的阶段过渡动画
- 📱 响应式布局

## 🚀 快速开始

### 运行方式
```bash
# 使用任意 HTTP 服务器（推荐）
python -m http.server 8000

# 访问 http://localhost:8000
```

> ⚠️ 请勿直接双击 `index.html` 打开，浏览器 CORS 限制可能导致资源加载失败。

### 操作指南
1. **浏览演化**：拖动底部时间轴，或点击 ▶️ 播放按钮自动播放
2. **查看详情**：右侧面板展示当前时期的物理描述与附件
3. **管理员模式**：点击左上角「管理员模式」按钮，输入密码 `zjuphy`
4. **编辑内容**：在管理员面板中编辑文档、上传附件、添加链接

## 📁 项目结构

```
universe_evolution/
├── index.html                 # 🏠 主入口（UI 框架 + 时间轴 + 管理员面板）
├── README.md                  # 📖 项目说明（本文件）
│
├── admin_uploads/             # 📁 管理员上传数据
│   ├── content.json           # 各时期文档内容与附件元数据
│   ├── images/                # 上传的图片文件
│   ├── videos/                # 上传的视频文件
│   └── docs/                  # 上传的文档文件
│
├── css/
│   └── style.css              # 自定义样式（时间轴、面板、滚动条等）
│
└── js/
    ├── main.js                # 核心逻辑（Three.js 场景、时间轴、UI 更新、管理员）
    ├── data.js                # 各演化阶段的配置数据（名称、时间、温度、描述）
    ├── shaders.js             # GLSL 着色器（顶点 + 片段着色器）
    └── stages/                # 🎆 各阶段的独立可视化模块
        ├── very_early/        # 极早期宇宙（能量主导、宇宙暴涨）
        ├── particle_formation/# 粒子形成（轻子夸克时期）
        ├── matter_creation/   # 物质诞生（核合成）
        └── structure_formation/ # 结构形成（恒星、星系）
```

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构 |
